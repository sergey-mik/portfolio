Projects

Random Winner - Front-End

This application will function as an online store, but with a twist: instead of paying full price, the product will be divided into ten open slots. Customers will be able to place bids on these slots. Once all the slots are filled, the application will randomly pick a winner. The staff will have the ability to add, edit, update, and delete products. The staff will also have access to all the user profiles and can activate or deactivate users as needed.

Ferry Ride - Full-Stack

This application is a ferry booking system that allows users to store and access data related to ferry rides. Users can view ferry schedules, reserve seats, save ferry departures, and leave comments. The application uses the data stored in the database to provide users with up-to-date and accurate information about available ferry rides, and to offer a seamless and user-friendly experience for booking their trips.
